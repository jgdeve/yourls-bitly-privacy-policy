Privacy Policy

YOURLS Bitly Client

Effective date: 11.02.2025

YOURLS Bitly Client (“the App”) is a URL shortening application that allows users to shorten links using self-hosted YOURLS instances and the Bitly API.

We respect your privacy and are committed to protecting it. This Privacy Policy explains how information is handled when you use the App.

Information We Collect

The App may collect the following information:

User-provided data:
URLs entered by the user for the purpose of shortening links.

How We Use Information

The information provided by the user is used only to:

Generate shortened URLs using YOURLS or Bitly APIs.

The App does not store URLs permanently on the device or on any external server owned by the developer.

Data Sharing

We do not sell, share, or rent user data to any third party.

URLs are transmitted only to the selected URL shortening service (YOURLS or Bitly) as required for functionality.

Data Security

All data transmission occurs over secure HTTPS connections.

No personal data is stored by the App.

Data Retention & Deletion

The App does not retain user data.

Since no data is stored, users can effectively delete their data by closing the App.

Children’s Privacy

This App is not intended for children under the age of 13.

The App does not knowingly collect personal information from children.

Third-Party Services

The App interacts with:

YOURLS (self-hosted URL shortener)

Bitly API

Use of these services is subject to their respective privacy policies.
